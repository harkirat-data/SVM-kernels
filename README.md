# SVM Kernels

This repository contains the implementation of Support Vector Machine (SVM) kernels using Python and Scikit-learn.

## Topics Covered
- Linear SVM
- Polynomial Kernel
- RBF Kernel
- Kernel Trick
- Feature Transformation
- Decision Boundary Visualization

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Plotly
- Scikit-learn
